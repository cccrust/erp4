# ERP4 v0.4

在 v0.3 輸出格式強化基礎上，新增資料匯入、JSON 輸出與資料完整性工具。

## 預計新增功能

### ☐ 1. CSV 資料匯入

新增 `import` 子指令，支援從 CSV 檔案批次建立客戶、產品、供應商。

```
erp4 customer import customers.csv
erp4 product  import products.csv
erp4 supplier import suppliers.csv
```

**CSV 欄位對應（標題列必須完全符合）：**

| 實體 | 必要欄位 | 可選欄位 |
|------|---------|---------|
| customer | `name` | `email`, `phone`, `address` |
| product | `name`, `sku`, `price` | `stock`, `description` |
| supplier | `name` | `contact_person`, `email`, `phone`, `address` |

- 支援 BOM（UTF-8 with BOM，Excel 匯出常見）
- 第一列為標題列，比對欄位名稱後依序讀取
- 逐列處理，單列錯誤不中斷整體匯入
- 匯入結束時顯示成功/失敗統計：`已匯入 47 筆，跳過 3 筆錯誤`
- 採用簡單自訂 CSV 解析器（不引入額外套件），僅需處理基本 quoting 與 escaping

### ☐ 2. JSON 輸出格式 (`--format json`)

所有 `list`、`get`、`report` 指令支援 `--format json`，輸出 JSON 陣列或物件。

```
erp4 customer list --format json
erp4 product  get  1  --format json
erp4 order    list --status confirmed --format json
erp4 report   sales --from 2026-01-01 --to 2026-06-30 --format json
```

- `list` 輸出 JSON 陣列
- `get` 輸出單一 JSON 物件
- `report sales` 輸出 `{ "rows": [...], "total_qty": N, "total_amount": N }`
- 輸出格式為縮排美化（pretty-print）
- 需新增 `serde_json` 相依，並為所有資料結構 derive `Serialize`

### ☐ 3. 匯出全部資料 (`erp4 export`)

新增 `export` 指令，將客戶、產品、供應商、訂單、採購單、發票全部匯出為單一 JSON 檔案。

```
erp4 export --format json > backup.json
erp4 export --format csv > backup.csv
```

- JSON 格式：`{ "customers": [...], "products": [...], ... }`
- CSV 格式：多個 CSV 區塊以註解行分隔
- 目的：備份、搬遷資料庫

### ☐ 4. `get` 指令強化

- 所有 `get` 指令（customer、product、supplier、order、purchase-order、invoice）支援 `--format json` 與 `--format table`（預設）
- 讓單一記錄可以程式化讀取

## 相依套件評估

| 套件 | 用途 | 評估 |
|------|------|------|
| `serde_json` | JSON 序列化 | 必要。與現有 `serde` 搭配，生態系標準 |
| 自訂 CSV 解析 | CSV 匯入 | 不引入額外套件。格式單純，手寫 parser 約 50 行 |

## 檔案變更預測

| 檔案 | 變更 |
|------|------|
| `Cargo.toml` | 新增 `serde_json` |
| `src/cli/*.rs` | 各 `get` 子指令新增 `--format`；所有 struct derive `Serialize` |
| `src/cli/mod.rs` | 新增 `Import`、`Export` 指令分派 |
| `src/cli/import.rs` | **新增** — CSV 匯入邏輯 |
| `src/cli/export.rs` | **新增** — 全部資料匯出 |
| `src/model/*.rs` | 所有資料結構 derive `Serialize` |
| `src/cli/fmt.rs` | 新增 JSON 格式化函式 |
