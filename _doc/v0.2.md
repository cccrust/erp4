# ERP4 v0.2

在 v0.1 基礎 CRUD 之上，強化庫存自動連動、搜尋篩選、報表、與測試覆蓋。

## 新增 / 改進功能

### ✅ 1. 庫存自動連動

| 事件 | 行為 | 狀態 |
|------|------|------|
| `order update-status <id> confirmed` | 扣減訂單中各項產品的 stock | ✅ |
| `order update-status <id> cancelled` | 還原 stock（若此前已扣） | ✅ |
| `purchase-order update-status <id> received` | 增加採購單中各項產品的 stock | ✅ |
| `purchase-order update-status <id> cancelled` | 還原 stock（若此前已入庫） | ✅ |

庫存不足時拒絕操作並顯示明確錯誤訊息。

### ✅ 2. 搜尋 / 篩選

```
customer list --search <keyword>        # 比對 name / email / phone
product  list --search <keyword>        # 比對 name / sku
product  list --low-stock <n>           # 庫存低於 n
order    list --status <status>         # 依狀態篩選
order    list --customer-id <id>        # 依客戶篩選
invoice  list --status <status>         # 依狀態篩選
invoice  list --customer-id <id>        # 依客戶篩選
purchase-order list --status <status>   # 依狀態篩選
```

### ✅ 3. 報表指令

```
erp4 report sales [--from YYYY-MM-DD] [--to YYYY-MM-DD]
    按產品別統計銷售數量與金額，含總計行

erp4 report inventory
    列出所有產品庫存、庫存價值 (stock × price)、低庫存警示 (*)

erp4 report aging
    應收帳款 / 未付款發票清單，含總計
```

### ✅ 4. 自動編號

- 新增 `sequences` 資料表
- `invoice create` 的 `invoice_number` 改為可選，不傳時自動產生
- 格式：`INV-2026-0001`

### ✅ 5. 資料驗證

| 驗證 | 說明 |
|------|------|
| `price` | 必須 > 0 |
| `stock` | 必須 >= 0 |
| `email` | 基本格式檢查（含 `@` 和 `.`） |
| `status` | 僅接受白名單中的值 |
| `amount` (發票) | 必須 > 0 |
| `quantity` (明細) | 必須 > 0 |

### ✅ 6. 單元測試

```
cargo test
```

18 個測試，覆蓋：

- **product:** CRUD、價格驗證、庫存調整、搜尋
- **customer:** CRUD、搜尋、Email 驗證
- **order:** CRUD、庫存連動（確認扣庫 / 取消還原）、庫存不足拒絕、狀態篩選
- **purchase_order:** CRUD、庫存連動（收貨入庫 / 取消還原）、狀態篩選
- **invoice:** CRUD、自動編號、自訂編號、金額驗證、狀態篩選、帳齡報表

### ✅ 7. CLI 參數調整

- `invoice create` 改為命名參數：`--customer-id <id> --due-date <date> --amount <n> [--order-id] [--notes]`
- 所有 `list` 指令新增篩選參數
