# ERP4 v0.3

在 v0.2 庫存連動、搜尋篩選與報表基礎上，強化輸出格式、使用者體驗與資料匯出。

## ✅ 新增功能

### ✅ 1. CSV 匯出 (`--format csv`)

所有 `list` 指令與 `report` 指令支援 `--format csv` 參數，輸出逗號分隔值。

```
erp4 customer list --format csv
erp4 product  list --format csv
erp4 order    list --status confirmed --format csv
erp4 report   sales --from 2026-01-01 --to 2026-06-30 --format csv
```

- 標題列使用英文欄位名稱（利於後續匯入其他系統）
- 純文字輸出、無裝飾線、無千分位

### ✅ 2. 輸出格式強化

- **金額千分位：** 所有 CLI 輸出金額加上逗號千分位（如 `61,970.00`）
- **著色輸出：** 使用 `colored` crate
  - 標題列：黃色粗體
  - 狀態：`cancelled` 紅色、`paid`/`delivered`/`received` 綠色、`confirmed`/`approved` 青色
  - 錯誤訊息：紅色粗體前綴 `Error:`
- `colored` 原生支援 `NO_COLOR` 環境變數

### ✅ 3. 分頁支援（選擇性）

- `list` 指令與 `report sales`、`report inventory` 支援 `--page` 與 `--page-size`
- SQL 層級 LIMIT/OFFSET 實現，不浪費記憶體
- 不指定時回傳全部（維持向下相容）

### ✅ 4. 錯誤訊息改善（部分）

- `create`、`update`、`update-status`、`add-item` 等指令統一使用 `Error: <msg>` 格式
- `product list`、`customer list` 等唯讀查詢維持原本「查無資料」風格
