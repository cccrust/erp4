# ERP4 v0.1

Rust + SQLite 打造的企業資源規劃 CLI 工具。

## 技術棧

- **語言:** Rust (edition 2021)
- **資料庫:** SQLite (via rusqlite, bundled)
- **CLI 框架:** clap (derive 模式)
- **日期處理:** chrono
- **錯誤處理:** anyhow

## 資料庫結構 (8 張表)

| 表名 | 說明 |
|------|------|
| `customers` | 客戶資料 (name, email, phone, address) |
| `products` | 產品資料 (name, sku, price, stock, description) |
| `suppliers` | 供應商資料 (name, contact_person, email, phone, address) |
| `orders` | 銷售訂單 (customer_id, order_date, status, total_amount, notes) |
| `order_items` | 訂單明細 (order_id, product_id, quantity, unit_price) |
| `purchase_orders` | 採購單 (supplier_id, order_date, status, total_amount, notes) |
| `purchase_order_items` | 採購單明細 (purchase_order_id, product_id, quantity, unit_price) |
| `invoices` | 發票 (invoice_number, order_id, customer_id, invoice_date, due_date, status, amount, notes) |

## CLI 指令

### 初始化

```
erp4 init
```

建立 `erp4.db` (可透過環境變數 `ERP4_DB` 自訂路徑)。

### 客戶管理

```
erp4 customer add <name> [--email] [--phone] [--address]
erp4 customer list
erp4 customer get <id>
erp4 customer update <id> [--name] [--email] [--phone] [--address]
erp4 customer delete <id>
```

### 產品管理

```
erp4 product add <name> <sku> <price> [--stock] [--description]
erp4 product list
erp4 product get <id>
erp4 product update <id> [--name] [--sku] [--price] [--stock] [--description]
erp4 product delete <id>
```

### 供應商管理

```
erp4 supplier add <name> [--contact-person] [--email] [--phone] [--address]
erp4 supplier list
erp4 supplier get <id>
erp4 supplier update <id> [--name] [--contact-person] [--email] [--phone] [--address]
erp4 supplier delete <id>
```

### 訂單管理

```
erp4 order create <customer-id> [--notes]
erp4 order list
erp4 order get <id>
erp4 order update-status <id> <status>
erp4 order delete <id>
erp4 order add-item <order-id> <product-id> <quantity> [--unit-price]
erp4 order items <order-id>
```

`status` 可選值: `pending`, `confirmed`, `shipped`, `delivered`, `cancelled`

### 採購單管理

```
erp4 purchase-order create <supplier-id> [--notes]
erp4 purchase-order list
erp4 purchase-order get <id>
erp4 purchase-order update-status <id> <status>
erp4 purchase-order delete <id>
erp4 purchase-order add-item <po-id> <product-id> <quantity> [--unit-price]
erp4 purchase-order items <po-id>
```

`status` 可選值: `pending`, `approved`, `received`, `cancelled`

### 發票管理

```
erp4 invoice create <invoice-number> --customer-id <id> <due-date> <amount> [--order-id] [--notes]
erp4 invoice list
erp4 invoice get <id>
erp4 invoice update-status <id> <status>
erp4 invoice delete <id>
```

`status` 可選值: `unpaid`, `paid`, `overdue`, `cancelled`

## 使用範例

```bash
# 初始化
erp4 init

# 新增客戶與產品
erp4 customer add "Acme Corp" --email info@acme.com
erp4 product add "Widget A" WDG-001 19.99 --stock 100

# 建立訂單並加入產品
erp4 order create 1 --notes "First order"
erp4 order add-item 1 1 10

# 查詢訂單 (含明細)
erp4 order get 1

# 產生發票
erp4 invoice create INV-001 --order-id 1 1 2026-07-11 199.90

# 更新狀態
erp4 order update-status 1 shipped
erp4 invoice update-status 1 paid
```

## 建置

```bash
cargo build --release
```

執行檔位於 `target/release/erp4`。
